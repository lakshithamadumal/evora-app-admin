# 🛠️ Evora Admin App

Evora Admin is a management application designed to handle products, orders, customers, and store operations efficiently.

## ✨ Features

* Product management
* Inventory management
* Order management
* Customer monitoring
* Sales overview
* Secure admin authentication

## 🛠️ Built With

* Android (Java)
* Firebase

## 🚀 Getting Started

```bash
git clone <git@github.com:lakshithamadumal/evora-app-admin.git>
```

Open the project in Android Studio and run the application.
